# Activity Minimization of Misinformation Influence in Online Social Networks

## About
This is a social networking web app where users can post content, 
add friends, and chat. An admin monitors and filters misinformation 
by blocking users who spread false information.

## Features
### User
- Register and login
- Add friends and accept requests  
- Create posts with images
- Like and comment on friends posts
- Chat with friends

### Admin
- Approve or reject new registrations
- View all posts and comments
- Add filter words to detect misinformation
- Block users automatically based on comments
- View results and charts

## How to Run
1. Make sure Python is installed
2. Open terminal in the project folder
3. Run: python app.py
4. Open browser and go to: http://127.0.0.1:8000
